## Correr el Proyecto Local

Para ejecutar el proyecto localmente, sigue estos pasos:

1. **Compilar e instalar las dependencias del proyecto**:
    ```bash
    mvn clean install
    ```

2. **Iniciar la aplicaci√≥n Spring Boot**:
    ```bash
    mvn spring-boot:run
    ```

3. **Generar el paquete del proyecto**:
    ```bash
    mvn clean package
    ```
